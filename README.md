# ⚡ JJK Domain Expansion Hub

Welcome to the **JJK Domain Expansion Hub** - an anime-inspired interactive web experience built to capture the intensity of Domain Expansion moments. 🌀🔥

## 🌟 Project Overview

This project delivers a cinematic frontend experience inspired by iconic cursed techniques, combining:

- high-impact visuals and transitions
- immersive scene design
- modern React + Vite performance
- smooth local development workflow

## ✨ Features

- ⚔️ JJK-inspired interactive experience
- 🎬 Cinematic UI transitions and effects
- ⚡ Fast frontend setup powered by Vite
- 🧩 Modular component-based architecture
- 📱 Responsive layout for different screen sizes

## 🧰 Tech Stack

- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Package Manager:** npm

## ✅ Prerequisites

Before running this project, make sure you have:

- `Node.js` (recommended LTS version)
- `npm` (comes with Node.js)

## 🚀 Getting Started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start the development server**

   ```bash
   npm run dev
   ```

3. **Open the local URL**
   - After the server starts, open the URL shown in your terminal (usually `http://localhost:5173`).

## 🛠️ Useful Commands

- `npm run dev` - start local development server
- `npm run build` - create production build
- `npm run preview` - preview production build locally

## 🎨 Customization Tips

- Update visuals and animation timing inside components in `src/components/`.
- Adjust app metadata in `metadata.json` and HTML metadata in `index.html`.
- Tune style tokens and utility classes to match your preferred theme.

## 🧪 Troubleshooting

- If dependencies fail to install, remove `node_modules` and run `npm install` again.
- If the port is busy, run with another port:
  ```bash
  npm run dev -- --port 5174
  ```
- If changes do not appear, restart the dev server.
