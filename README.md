# JJK-Fun-Gesture

An immersive experience featuring Gojo Satoru's Unlimited Void, Cursed Technique Reversal: Red, and Sukuna's Malevolent Shrine with real-time hand tracking.

## Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/2d5ad893-59cc-4b48-8c37-3a8b02e8b791

## Run locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app: `npm run dev`
