# Plan: Add favicon and improve meta tags

## Goal
Move favicon.ico to the correct folder and update HTML metadata for better SEO.

## Steps
1. **Move favicon.ico**: Copy `favicon.ico` from root to `public/` folder
2. **Add favicon link**: Add `<link rel="icon" href="/favicon.ico">` to index.html head
3. **Improve meta tags**: Add additional SEO improvements:
   - Add `theme-color` meta tag
   - Add `robots` meta tag
   - Add favicon type for better compatibility

## Implementation
- Copy: `favicon.ico` → `public/favicon.ico`
- Edit `index.html`:
  - Add favicon link in `<head>`
  - Add `<meta name="theme-color" content="#1a1a2e">` for browser theme
  - Add `<meta name="robots" content="index, follow">`
